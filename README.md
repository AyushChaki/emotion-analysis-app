# 🎭 Emotion Analysis App

An end-to-end **Machine Learning + NLP** project that analyzes text and predicts the underlying human emotion using Natural Language Processing techniques and classification models.

This project is built with **Python**, **Scikit-learn**, and **Streamlit**, and provides a simple interactive web interface where users can enter text and instantly receive emotion predictions.

# 🧠 How It Works

1. User enters a sentence/text.
2. The text is cleaned and preprocessed.
3. TF-IDF vectorization converts text into numerical format.
4. The trained ML model predicts the emotion.
5. Result is displayed instantly on the Streamlit app.

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/AyushChaki/emotion-analysis-app.git
```

## 2️⃣ Navigate to the Project Folder

```bash
cd emotion-analysis-app
```

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## 4️⃣ Run the Streamlit App

```bash
streamlit run app.py
```
