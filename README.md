🎭 Emotion Analysis App

An end-to-end Machine Learning + NLP project that analyzes text and predicts the underlying human emotion using Natural Language Processing techniques and classification models.

This project is built with Python, Scikit-learn, and Streamlit, and provides a simple interactive web interface where users can enter text and instantly receive emotion predictions.🚀 Live Demo

How does it work?
User enters a sentence/text.
The text is cleaned and preprocessed.
TF-IDF vectorization converts text into a numerical format.
The trained ML model predicts the emotion.
The result is displayed instantly on the Streamlit app.
